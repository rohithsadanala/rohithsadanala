*Engineered highly scalable LLM agent systems capable of multi-step reasoning, dynamic tool execution, and self-correction.*
* **Architecture:** LangGraph/LangChain orchestration pipelines coupled with comprehensive LLM evaluation and monitoring.
* **Impact:** Scaled backend systems to support **50K+ RPS** while maintaining **sub-200ms inference latency**.

### 📉 Hybrid Fraud Detection Engine
*Designed a hybrid ML + LLM classification system for real-time refund fraud detection.*
* **Architecture:** CLIP + FAISS for visual similarity combined with LLM-based anomaly reasoning.
